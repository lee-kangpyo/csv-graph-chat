## Purpose
바구니 항목 클릭 시 그래프를 화면의 80% 크기 모달 오버레이로 표시하는 시스템

## Requirements

### Requirement: 모달 오버레이로 그래프 표시
시스템은 사용자가 BasketItem을 클릭하면 저장된 그래프를 화면 너비·높이 각 약 80% 크기의 모달 오버레이에 표시해야 한다.

#### Scenario: BasketItem 클릭으로 그래프 보기
- **WHEN** 사용자가 사이드바에서 BasketItem을 클릭할 때
- **AND** 해당 항목에 유효한 `graph_config`가 존재할 때
- **THEN** 화면 중앙에 모달 오버레이가 표시된다
- **AND** 모달 크기는 화면 너비·높이의 약 80%이다
- **AND** 모달 안에 그래프가 충분한 크기로 렌더링된다
- **AND** 해당 그래프를 만든 사용자 질문이 함께 표시된다
- **AND** 배경은 어둡게 처리된다
- **AND** 나머지 화면은 오버레이로 가려진다

#### Scenario: 로딩 placeholder 클릭 시 모달 금지
- **WHEN** BasketItem이 로딩 placeholder 상태(`graph_config` 없음)일 때
- **THEN** 클릭해도 모달 오버레이는 열려서는 안 된다

### Requirement: 모달 닫기
시스템은 사용자가 모달을 닫고 본 화면으로 돌아갈 수 있는 수단을 제공해야 한다.

#### Scenario: 닫기 버튼
- **WHEN** 사용자가 모달 헤더의 닫기(X) 버튼을 클릭할 때
- **THEN** 모달이 닫힌다
- **AND** 사용자는 메인 화면으로 돌아간다
- **AND** 바구니 사이드바는 계속 사용할 수 있다

#### Scenario: 오버레이 클릭으로 닫기
- **WHEN** 사용자가 모달 콘텐츠 바깥의 오버레이를 클릭할 때
- **THEN** 모달이 닫힌다
- **AND** 사용자는 메인 화면으로 돌아간다

### Requirement: 사용자 질문 맥락 표시
시스템은 그래프를 생성하게 한 사용자 질문을 모달 헤더 영역에 표시해야 한다.

#### Scenario: 모달에 질문 표시
- **WHEN** 그래프 모달이 열릴 때
- **THEN** 모달 헤더에 사용자 질문 텍스트가 보인다
- **AND** 글자 크기는 최소 14px로 읽기 쉽다
- **AND** 질문이 길면 모달 너비에 맞게 줄바꿈된다
- **AND** 질문은 시각화 위쪽에 배치된다

### Requirement: 모달 내 ECharts 렌더링
시스템은 모달 안의 그래프를 저장된 설정을 유지한 채 ECharts로 렌더링해야 한다.

#### Scenario: ECharts로 그래프 표시
- **WHEN** 모달이 그래프 데이터와 함께 열릴 때
- **THEN** 저장된 `graph_config`로 ECharts 인스턴스가 초기화된다
- **AND** 그래프 크기는 모달(약 80% 화면)에 맞게 조정된다
- **AND** 모달 크기 변경에 반응해 그래프가 조정될 수 있다
- **AND** 원래 스타일·설정이 가능한 범위에서 유지된다
