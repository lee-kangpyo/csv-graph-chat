## MODIFIED Requirements

### Requirement: 미리보기 클릭 동작
#### Scenario: 미리보기 클릭 시 전체 모달
- **WHEN** 바구니 항목이 **로딩 placeholder** 상태(`graph_config`가 아직 없음)일 때
- **THEN** 사용자가 항목(미리보기 포함)을 클릭해도 전체 그래프 모달은 열려서는 안 된다
- **AND** 로딩 상태임을 사용자가 인지할 수 있는 UI(스피너/문구)가 유지되어야 한다

- **WHEN** 바구니 항목이 최종 저장되어 `graph_config`가 존재할 때
- **THEN** 사용자가 그래프 미리보기 썸네일을 클릭할 때
- **THEN** 화면 약 80% 크기의 전체 그래프 모달이 열린다
- **AND** 모달에는 완전한 그래프와 사용자 질문이 표시된다
- **AND** 동작은 항목 텍스트 영역 클릭과 동일하다
