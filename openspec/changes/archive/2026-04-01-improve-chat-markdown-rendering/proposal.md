## Why

현재 ChatArea의 Message 컴포넌트가 AI 응답을 **평문으로만** 렌더링합니다. 결과적으로:

- 마크다운 헤더(`###`, `**`)이 그대로 출력됨
- 코드 블록이 이쁘게 표현되지 않음
- 목록이 들여쓰기 없이 출력됨
- AI가 상세한 분석을 제공해도 **가독성이 떨어짐**

사용자가 긴 분석 메시지나 코드, 목록 등을 확인할 때 의미 단위로 파악이 어려워 불편합니다.

## What Changes

**프론트엔드:**
- `react-markdown` 라이브러리 설치
- `Message.jsx`에서 마크다운 콘텐츠를 이쁘게 렌더링
- 코드 블록에 구문 강조 적용 (선택적)
- 링크 클릭 가능, 목록 들여쓰기, 헤더 계층 구조 적용

**不影响:**
- 백엔드 API 변경 없음
- AI 응답 형식 변경 없음
- 기존 기능 변경 없음

## Capabilities

### Modified Capabilities

- `ai-graph-conversation`: AI 응답의 가독성이 향상되어 사용자가 분석 결과를 더 쉽게 이해할 수 있음
