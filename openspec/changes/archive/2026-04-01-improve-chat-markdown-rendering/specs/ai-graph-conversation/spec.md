## MODIFIED Requirements

### Requirement: 실시간 스트리밍 응답
#### Scenario: AI 응답 메시지가 마크다운으로 렌더링됨
- **WHEN** AI가 마크다운 형식의 응답을 생성할 때
- **THEN** Message 컴포넌트는 마크다운 요소를 이쁘게 렌더링해야 한다
- **AND** 헤더(h1-h3)가 크기별로 구분되어 표시된다
- **AND** 코드 블록이 적절한 스타일로 표시된다
- **AND** 목록이 들여쓰기와 불릿/번호로 표시된다
- **AND** 링크가 클릭 가능해야 한다
- **AND** 강조(strong, em)가 적절히 표현되어야 한다

#### Scenario: 사용자 메시지는 기존대로 평문 유지
- **WHEN** 사용자가 메시지를 보낼 때
- **THEN** 메시지는 마크다운 처리 없이 평문으로 표시된다
